PEP 8 Style for Production Code

Production code is read far more often than it is written. This function shows the habits reviewers look for: a docstring, snake_case names, 4-space indentation, and a type hint.

PEP 8 in practice — a clean ETL function
import csv

def load_transactions(file_path: str) -> list:
    """Extract banking transactions from CSV for nightly load."""
    records = []
    with open(file_path, "r") as f:
        for row in csv.DictReader(f):
            records.append(row)
    return records

txns = load_transactions("transactions.csv")
print(len(txns))    # Output (example): 14230 — rows loaded

Understanding SELECT Queries — My SQL Breakthrough

sql learning

The Mental Model

For weeks I treated SQL like a programming language. The breakthrough came when I started thinking of it as asking questions of a table.

"Show me all employees with salary over 50000."

That maps directly to:

SQL
-- Show all employees with salary greater than 50000
SELECT employee_id, first_name, salary FROM hr.employees WHERE salary > 50000;

Practice Pattern

  1. Write the question in plain English
  2. Identify the table
  3. Add filters with WHERE
  4. Choose specific columns instead of * when you can
